Cómo subir páginas a GitHub Pages

En esta guía paso a paso, te llevaré a través del proceso de subir tu sitio web a GitHub Pages, un servicio de alojamiento gratuito ofrecido por GitHub. Esto nos permitirá ofrecer nuestra web de forma pública a cualquier persona en la web de una forma sencilla.

Cualquier página que queramos alojar en GitHub Pages deberá ser un repositorio. Además, si no tenemos un plan Pro, deberá ser un repositorio público.

Entonces, lo primero que debemos hacer es crear un repositorio donde tengamos nuestra página. Si ya tenemos nuestra página en un repositorio, nos debemos asegurar de que este repositorio sea público.

Para activar GitHub Pages, iremos a la pestaña de configuración del repositorio, y en la sección de Pages, abriremos el desplegable de "Build and deployment". Ahí simplemente seleccionamos la rama principal de nuestro repositorio, que será la que vayamos a desplegar.

GitHub Pages nos permitirá desplegar (deploy) una rama en concreto de nuestro repositorio. Podemos crear una rama "gh-pages", o podemos usar la rama por defecto (generalmente main), o cualquier otra. Esto nos permitirá crear ramas paralelas para realizar cambios o pruebas sin afectar a la web pública, y más tarde fusionarlos a la principal cuando todo funcione correctamente.

Con esto nuestra página ya estaría pública bajo la dirección (nuestrousuario).github.io/(nombre-repositorio), aunque puede tardar un rato en actualizarse.

Además, podemos crear una página de inicio para nuestro usuario, que usará la dirección (nuestrousuario).github.io. Para esto, simplemente crearemos nuestra página en un repositorio llamado igual que dicha url (nuestrousuario).github.io, y ya podremos usarla como página de inicio.

En este último caso estaríamos creando nuestra página principal en nuestro perfil, mientras que en el primer ejemplo estaríamos publicando una página para un repositorio en concreto. Esto es útil si queremos hacer una página para un proyecto (de presentación por ejemplo), o simplemente publicar una página individual.
